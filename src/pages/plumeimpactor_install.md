# PlumeImpactor 安装

**适用平台**：macOS / Windows  
**准备项**：PlumeImpactor、Apple ID、数据线

## 1. 安装 PlumeImpactor
1. 下载并安装 PlumeImpactor。
2. 启动后确认应用正常运行。

## 2. 连接设备
1. 使用 USB 连接 iPhone。
2. 在 iPhone 上点按“信任此电脑”。

## 3. 导入 IPA 并安装
1. 在 PlumeImpactor 中选择 IPA 文件。
2. 输入 Apple ID 与密码完成签名。
3. 等待安装完成。

## 4. 信任证书
1. iPhone 打开 设置 -> 通用 -> VPN 与设备管理。
2. 在开发者 App 下找到你的 Apple ID。
3. 点按“信任”。

## 常见问题
- 若安装失败，尝试更换数据线或 USB 口。
- 证书相关报错通常与 Apple ID 限制有关。
