# LiveContainer 安装

**适用平台**：iOS  
**准备项**：已侧载的 LiveContainer 安装包

## 1. 安装 LiveContainer
1. 通过任意侧载工具（如 AltStore 或 SideLoadly）安装 LiveContainer 的 IPA。
2. 在设备上打开 LiveContainer 进行首次配置。

## 2. 导入 IPA
1. 在 LiveContainer 中选择导入。
2. 通过 文件 App 或共享菜单选择 IPA。
3. 等待导入完成。

## 3. 运行与管理
1. 在 LiveContainer 内启动应用。
2. 可在列表中管理已导入的 IPA。

## 常见问题
- 若无法导入，确认 IPA 未损坏且文件路径可访问。
- 如遇权限提示，检查系统设置中的相关授权。
