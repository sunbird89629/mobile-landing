# SideLoadly 安装

**适用平台**：macOS / Windows  
**准备项**：Apple ID、SideLoadly、数据线

## 1. 下载与安装
1. 访问 SideLoadly 官网下载对应系统版本。
2. 安装完成后启动 SideLoadly。
3. 若 macOS 提示无法打开，可参考 [SideLoadly 无法打开教程](/side_loadly_open)。

## 2. 连接设备
1. 使用 USB 连接 iPhone。
2. 在 iPhone 上点按“信任此电脑”。

## 3. 导入 IPA 并安装
1. 点击 Browse 选择 IPA 文件。
2. 输入 Apple ID 与密码。
3. 点击 Start 开始安装。

## 4. 信任证书
1. iPhone 打开 设置 -> 通用 -> VPN 与设备管理。
2. 在开发者 App 下找到你的 Apple ID。
3. 点按“信任”。

## 常见问题
- Apple ID 限额可能导致安装失败，需等待名额释放。
- 安装卡住时，检查数据线连接或更换 USB 口。
